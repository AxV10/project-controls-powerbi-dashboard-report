# Project Controls Power BI Dashboard

## Overview

Designed, developed, and maintain a Power BI dashboard used for weekly executive project reporting on a large, multi-year infrastructure project. The dashboard consolidates data from multiple project sources to provide real-time visibility into project performance for leadership and stakeholders.

This dashboard serves as the primary reporting tool for weekly project updates distributed to project managers, directors, executives, owners, and clients.

> **Note:** All project data and screenshots in this repository have been anonymized to protect confidential information.

---

## My Role

I am responsible for the ongoing development and maintenance of this dashboard, including:

- Designing and improving dashboard visualizations
- Integrating data from multiple project data sources
- Building and maintaining Power Query transformations
- Creating DAX measures and KPIs
- Validating data accuracy before weekly reporting
- Implementing dashboard enhancements based on stakeholder feedback
- Supporting weekly executive reporting throughout the life of the project

---

## Dashboard Features

- Executive project overview
- Engineering progress tracking
- Procurement status reporting
- Construction progress monitoring
- Interactive filters and drill-through analysis
- Weekly KPI reporting
- Automated data refresh workflow
- Real-time project visibility
